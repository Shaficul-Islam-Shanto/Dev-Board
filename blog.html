Question-1: What are the different ways to select an element in the DOM?

In JavaScript, you can select elements in the DOM using different methods. getElementById(id) selects an element by its ID, getElementsByClassName(className) selects elements by class name, and getElementsByTagName(tagName) selects elements by their tag name. querySelector(selector) selects the first matching element, while querySelectorAll(selector) selects all matching elements.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML sets the HTML content, including tags. innerText or only the visible text. textContent sets the text content without considering CSS styles, and it’s faster than innerText as it doesn’t trigger reflows.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where you attach a single event listener to a parent element, and use the event's target to determine which child element clicked the event. This improves performance and handles dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event applied on a child element propagates upwards to its parent elements in the DOM tree, allowing parent elements to handle events from their children.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, you can create elements using document.createElement(), add them to the DOM with appendChild(), and remove them using removeChild(). These methods help dynamically modify the page content.
